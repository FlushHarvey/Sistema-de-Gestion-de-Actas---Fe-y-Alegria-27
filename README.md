# Sistema-de-Gestion-de-Actas---Fe-y-Alegria-27
Sistema de gestión de actas PDF para instituciones educativas. Permite simular y renombrar automáticamente archivos según reglas predefinidas, con procesamiento paralelo, control de ejecución, barra de progreso y registro de eventos. Interfaz moderna en Python/CustomTkinter con soporte claro/oscuro.
